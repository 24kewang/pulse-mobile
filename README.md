# pulse-mobile

Basic Expo app scaffold for Pulse.

## Quick start

```bash
npm install
npm run start
```

## Structure

- `App.tsx` app entry
- `src/navigation` navigation setup
- `src/screens` schedule, scan, and more tabs
- `src/data` mock schedule data

## Environment

Set `EXPO_PUBLIC_API_BASE_URL` to point at the Express backend.
